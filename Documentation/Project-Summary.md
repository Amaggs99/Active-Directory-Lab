\# Project Summary



\## Objective



Deploy a functional Active Directory environment using Windows Server 2022 and Windows 11.



\## Environment



\- Domain Controller: DC01

\- Client Workstation: CLIENT01

\- Domain: adlab.local

\- Network: VMnet1 Host-Only

\- DC01 IP Address: 192.168.66.10

\- CLIENT01 IP Address: 192.168.66.20



\## Completed Work



\- Installed Windows Server 2022

\- Renamed server to DC01

\- Installed Active Directory Domain Services

\- Promoted DC01 to Domain Controller

\- Configured DNS

\- Created Organizational Units

\- Created Security Groups

\- Created Test Users

\- Added users to security groups

\- Configured Windows 11 client networking

\- Joined CLIENT01 to adlab.local

\- Verified domain authentication

\- Captured screenshots for documentation



\## Skills Demonstrated



\- Windows Server Administration

\- Active Directory Administration

\- DNS Configuration

\- Network Troubleshooting

\- Domain Joining

\- User and Group Management

\- PowerShell Administration

\- Git and GitHub Documentation



\## Final Status



This repository documents the setup and deployment of the Active Directory environment. Future help desk scenarios, Group Policy management, and PowerShell automation will be documented in separate repositories.
